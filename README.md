# Project Overview
This project is a Food Delivery application that streamlines the process of ordering food from local restaurants.

# Tech Stack
- **Frontend:** Vue.js, CSS, SCSS
- **Backend:** PHP

# Project Structure
- **/frontend** - Contains all frontend files.
- **/backend** - Contains all backend files.
- **/restaurant-module** - Contains restaurant specific files.

# Key Features
- User-friendly interface for customers
- Restaurant management system
- Order tracking and notification system

# Getting Started
## Frontend
1. Clone the repo: `git clone https://github.com/Aryansinha123/FoodDelivery`
2. Navigate to the frontend directory: `cd frontend`
3. Install dependencies: `npm install`
4. Start the application: `npm run serve`

## Backend
1. Navigate to the backend directory: `cd backend`
2. Install dependencies

## Restaurant Module
- Instructions on how restaurants can manage their profiles and orders.
